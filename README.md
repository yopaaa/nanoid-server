# Foobar

simple server to test nodejs apps

demo result app
![demo ui](./ui.png)

## Installation

Use the package manager [npm]() to install this.

```bash
git clone https://github.com/yopaaa/nanoid-server.git
```

## Usage

```bash
# you can change port in .env
# default port is 8000

# install node modules
npm install

# run the server
npm run start

```
